# TeleMedBot

This project contains three main components:

1. **Backend**: NestJS application handling API, Bot logic, and Database interactions.
2. **Frontend**: React + Vite application for the Telegram Mini App.
3. **Contracts**: Solidity smart contracts for payment handling.

## Setup

1. Install dependencies in each directory:
   ```bash
   cd backend && npm install
   cd frontend && npm install
   cd contracts && npm install
   ```
2. Configure environment variables (.env) in backend.
3. Start backend: `cd backend && npm run start:dev`
4. Start frontend: `cd frontend && npm run dev`
